hexo怎么更换自己喜欢的主题

2019-01-02

2019新的开始，随心所欲，设置自己喜欢的主题

landscape是hexo的默认主题，才搭建好的时候看着还顺眼，到后面越看越不顺眼，于是下定决心改一下主题，纠结了好久，终于选到了自己喜欢的主题Ocean。

在这里我非常感谢数学家 的帮助。如果你喜欢这个主题，你可以点击此链接现在主题。

1、从Github上拉取Ocean的主题，在git执行命令行

git clone https://github.com/zhwangart/hexo-theme-ocean.git

2、你会获取一个Ocean的文件，到本地自己的hexo博客里面，找到themes主题配置文件，删除landscape文件夹，然后把你下载的Ocean文件夹拖进去就可以了。如果没有Ocena文件夹，自己创建一个，把你下载的文件拷贝进去。

3、然后在git里面执行以下命令行。

Ctrl+c  

hexo s

4、在本地你就可以看到你的主题换掉了。恭喜你换了你喜欢的主题。

5、修改你的配置

# Menu
//菜单配置
menu:
  主页: /
  归档: /archives
  相册: /Photo
  关于: /About


# Miscellaneous
favicon: /favicon.ico
brand: /blog/images/shark.svg

# Ocean Video
# Because I put videos in multiple formats on the same path, I just labeled the path here.
ocean_video_poster: http://source.zhwangart.com/hexo/themes/ocean/video/
ocean_video_src: http://source.zhwangart.com/hexo/themes/ocean/video/
ocean_video_brand: /blog/images/shark-inverted.svg

# Content
excerpt_link: 阅读全文...
nav_prev_text: 前一篇
nav_next_text: 后一篇
share_text: 分享

# fancybox
fancybox: true

# RSS
feed:
  type: atom
  path: atom.xml
  limit: 20
  hub:
  content:
  content_limit: 140
  content_limit_delim: ' '
  order_by: -date

# Local search
search_text: 搜索
# Dependencies: https://github.com/flashlab/hexo-generator-search
local_search:
  enable: false
  # if auto, trigger search by changing input
  # if manual, trigger search by pressing enter key or search button
  trigger: auto
  # show top n results per article, show all results by setting to -1
  top_n_per_article: 1
  unescape: true

# Gitalk
gitalk:
  enable: false//这个评论是我后面研究的，如果你希望添加评论功能，可以移步数学家的博客，他那边写的比较好。
  clientID: cd02b9326f9b90293627
  clientSecret: 8ff3d57e2fd26e9c1b9235aa0925e3e3a5363335
  repo: blog
  owner: mwm0522
  admin: mwm0522

6、如果以上配置完成，请在git里面执行以下命令行。

hexo clean

hexo g

hexo d

7、打开你线上的博客，你会发现你的博客主题变成了自己喜欢的主题。
